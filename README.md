# Convenia GraphQLClient

---

## Instalação

### Composer

Para instalar adicione o seguinte código em seu ```composer.json```

```json
{
    "require": {
        "convenia/graphqlclient": "^0.1"
    },
    "repositories": [
        {
            "type": "git",
            "url": "git@bitbucket.org:convenia/rh/libs/graphqlclient.git",
            "no-api": true
        }
    ]
}

```

Depois execute ```composer update```
